获取磁盘相关信息

#### 1 、diskinfo.getDrives(callback)
功能 ：获取当前已挂载磁盘内存相关信息

callback 回调参数，带有两个参数如：callback (err, aDrives)
err 0表示获取成功，-1表示获取失败
aDrives是一个对象数组，对象具有以下属性
| 成员 | 功能 |单位 |
| --- | ---| ---|
|aDrives[0].mounted|磁盘名(lfs,fatfs)| -|
|aDrives[0].blocks|磁盘总内存| kb |
|aDrives[0].used|磁盘已用内存| kb |
|aDrives[0].available|磁盘可用内存| kb|
|aDrives[0].capacity|磁盘已用内存百分比| %|
#### 2、示例
```javascript
var diskinfo = require('diskinfo');
diskinfo.getDrives(function (err, aDrives) {
    for (var i = 0; i < aDrives.length; i++) {
         	console.log('磁盘：' + aDrives[i].mounted);
         	console.log('磁盘总内存：' + aDrives[i].blocks);
        	console.log('磁盘已用内存：' + aDrives[i].used);
        	console.log('磁盘可用内存：' + aDrives[i].available);
        	console.log('磁盘已用内存百分比： ' + aDrives[i].capacity);
    }
})
```
