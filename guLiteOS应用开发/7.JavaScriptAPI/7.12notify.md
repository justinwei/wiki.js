# 一、参数以及api描述
### API 介绍
#### notify.createNotify()
创建的通知函数，并始终返回一个 currnotify对象。

#### currnotify 事件
当创建成功后，返回的 currnotify  对象可通过 on 方法监听多个事件，业务逻辑可在监听的回调函数中完成。以下列举一些常用的事件：
* error
##### 当通知失败时触发，参数为 错误值
```
currnotify.on('error', function(options) {
    console.debug('error');
});
```
* return
##### 返回时触发，参数为 成功失败
```
mediaPlayer.on('return', function(options) {
     console.debug('return');
});
```

#### currnotify 方法
currnotify 除监听事件外，也内置一些方法，以下列举一些常用的方法。

* ```currnotify.send(name, option)```
发送通知命令原生app执行相对应的功能
    * option：原生app需要的参数
      * name: 字符串，功能名称

### 例子
```
var notify= require('notify');

var curr_notify= notify.createNotify();
curr_notify.send('ota', option);

curr_notify.on('error', function(options) {
  console.debug('error cb');
});

mediaPlayer.on('return', function(options) {
  console.debug('return cb');
});

```
