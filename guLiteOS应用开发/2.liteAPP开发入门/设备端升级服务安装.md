
# 一、 升级服务安装：

##  1. 安装 nodejs 按以下地址
https://nodejs.org/zh-cn/download/

## 2. 打开 local_upgrage_server/service.js 文件
修改：
```
const local_ip = "192.168.1.106" //本机IP地址
const local_lfs_path = "/Users/weigang/workspace/Gu_lite_App/lv_sim_vscode_sdl"  //本机lfs 所在的绝对路径
```

## 3. 第一次安装组件
```
npm install
```

## 4. 启动服务
```
node service.js
```
