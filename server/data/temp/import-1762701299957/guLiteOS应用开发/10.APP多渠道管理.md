# 1. 渠道资源文件说明

# 1.1 文件目录说明
在APP的目录下，存在channel_resouce资源目录，目录结构如下：

```
├── channel_resouce                     渠道目录
         ├── standard.                       渠道名称目录                        
                  ├──  images                         当前渠道图片目录
                  ├── audio                            当前渠道音频目录
                  ├── resouce.json                  当前渠道资源文件
                  ├── app_default.less              当前渠道默认less
                  ├── drv_cfg                          当前渠道的驱动
         ├── pandan.                       渠道名称目录                        
                  ├──  images                         当前渠道图片目录
                  ├── audio                            当前渠道音频目录
                  ├── resouce.json                  当前渠道资源文件
                  ├── app_default.less              当前渠道默认less
                  ├── drv_cfg                          当前渠道的驱动
```
# 2. 分发路径

程序完成APP打包后，会自动生成打包后的文件 (这个文件提供给APP升级）
```
├── release                     发布目录
         ├── standard.                       渠道名称目录     
                 ├──  math_liteapp         子APP目录
                         ├──  code         代码文件(合并文件）
                         ├──  assets        资源目录   
                                ├──  images                         图片文件（合并文件）
                                ├── audio                            音频文件（合并文件）
                        ├──  sync.json    同步文件（用于升级）
                        ├──  drv_cfg.      驱动文件
                        ├──  font.bin.      字库文件
                  
         ├── pandan.                       渠道名称目录        
```
