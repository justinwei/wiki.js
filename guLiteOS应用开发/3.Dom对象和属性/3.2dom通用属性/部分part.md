不同的dom对象，可以通过定义不同part的样式实现不同的外观。
* 一个dom并非具备所有的部分

常用的part如下：
| 属性值 | 描述 |
|:-----|:----|
| part_main|主体部分|
| PART_SCROLLBAR|可滚动部分|
| PART_INDICATOR|聚焦部分|
| PART_KNOB|手柄部分|
| PART_SELECTED|选中部分|
| PART_ITEMS| 多个类似的元素，如单元格 |
| PART_TICKS| 刻度上的刻度，如图标或仪表 |
| PART_CURSOR| 标记一个特定位置，如文本域或图标的光标 |
| PART_CUSTOM_FIRST| 自定义部分 |
| PART_ANY|  |
