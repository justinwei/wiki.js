### 一、API 介绍

#### battery.createBattery()

创建并返回一个curr_battery对象。

### 二、curr_battery方法

**1、curr_battery.get()**

获取battery信息，返回一个options对象。

- options ：一个对象包括charge，full，init，val。
  - charge ：是否充电。
  - full ：是否满电。
  - init ：是否init。
  - val ：电量值。

### 三、事件

**1、battery_changed**

每过5秒触发一次battery_changed事件，返回一个options对象，参考get方法。

```js
curr_battery.on('battery_changed', function(options) {
    console.log(options);
  });
```



### 四、例子

```js
var battery = require('battery');

var curr_battery = battery.createBattery();

curr_battery.on('battery_changed', function(options) {
    console.log(options);
  });

var info = curr_battery.get();
console.log(info);

```
