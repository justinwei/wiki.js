### 一、模块介绍
一个缓冲区对象，该对象可用来申请一块内存，存放二进制数据。该缓冲区对象可直接使用nodejs提供的方法，可以查看https://nodejs.org/docs/v0.12.1/api/buffer.html
### 二、api
#### 1、Buffer(size)
分配 size 个字节的新Buffer
* size：申请缓冲区所需的长度
  
#### 2、Buffer(array)
拷贝array的新Buffer
* array：数组

#### 3、Buffer(buffer)
拷贝参数 buffer 的数据到新Buffer

#### 4、Buffer(str[, encoding])
返回一个被string的值初始化的新的Buffer
* string：字符串
* encoding：可选参数，string的编码，默认为"utf8"

#### 5、Buffer.isBuffer(obj)
如果 obj 是 Buffer，则返回 true，否则返回 false
* obj: object
* 返回：bool

#### 6、Buffer.isEncoding(encoding)
如果 encoding 是支持的字符编码的名称，则返回 true，否则返回 false， 目前仅支持utf8、base64、hex编码
* encoding: 要检查的字符编码名称
* 返回：bool

#### 7、Buffer.byteLength(string, [encoding])
提供字符串的实际字节长度
* string: 字符串
* encoding: 字符串，可选，默认值：“utf8"
* 返回：长度

#### 8、Buffer.concat(list, [totalLength])
合并缓冲区，该缓冲区是连接列表一起
* list: 用于合并的 Buffer 对象数组列表
* totalLength: 指定合并后 Buffer 对象的总长度
* 返回：返回一个多个成员合并的新 Buffer 对象

#### 8、Buffer.compare(buf1, buf2)
通常用于对实例数组进行排序的目的进行比较
* buf1: 缓冲区
* buf2: 缓冲区
* 返回：一个整数 0:相等，-1: buf1< buf2,  1:buf1>buf2

### 三、例子

```
var buf = new Buffer(20);
console.log(buf);
console.log(buf.length);
buf.write("some string");
console.log(buf);
console.log(buf.length);
buf = buf.slice(0,5);
console.log(buf);
console.log(buf.length); // 5

var buf = new Buffer([1, 2, 3]);
console.log(buf);
console.log(buf.length);

var buf = new Buffer("helloworld");
var buf1 = new Buffer(buf);
console.log(buf);
console.log(buf1);


var buf1= new Buffer(26);
for (var i = 0 ; i < 26 ; i++) {
  buf1[i] = i + 97; // 97 is ASCII a
}
console.log(buf1.toString('utf8')); // outputs: abcdefghijklmnopqrstuvwxyz
console.log(buf1.toString('utf8', 0, 5));      //abcde
console.log(buf1.toString('base64', 0, 5));     //YWJjZGU=

var str = '\u00bd + \u00bc = \u00be';
console.log(str + ": " + str.length + " characters, " +
  Buffer.byteLength(str, 'utf8') + " bytes");


var arr1 = [Buffer('1234'), Buffer('4567')];
console.log(Buffer.concat(arr1));

var arr = [Buffer('1234'), Buffer('0123')];
console.log(arr.sort(Buffer.compare));

//base64解码
var buf = new Buffer("aGVsbG93b3JsZA==", "base64");
console.log(buf);
console.log(buf.toString());
    
var buf2 = new Buffer(30);
buf2.write("aGVsbG93b3JsZA==", 0, 16, "base64");
console.log(buf2);
console.log(buf2.toString());
```
