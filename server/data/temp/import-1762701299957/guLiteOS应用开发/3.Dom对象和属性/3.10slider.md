* 滑动条，常用于音量调节、亮度调节等滑动场景
* 适配了触屏和原生按键导航

## 快速开始
![图片](../../image/a80e1710-c93b-4e96-95e2-b6f4ba2c7c1c.png)

```
<div class="container">
    <slider value="30" max_value="100" class="slider"align="center" @value_changed="changeCb()"></slider>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.slider {
    height: 20;
    width: 200;
    bg_color: #BEBEBE;
    radius: 20;

    &:PART_INDICATOR {
        bg_color: #00FF00;
    }

    &:PART_KNOB {
        bg_color: #FF00FF;
    }
}

changeCb(val) {
    logger.log('changeCb,val=', val);
},
```

## 常用属性
**只能定义在dom上。**
| 属性名 | 描述 | 值类别 |
|:-----|:----|:----|
| value | 滑动条初始值 | int |
| max_value | 滑动条最大值 | int |
| min_value | 滑动条最小值 | int |
| mode | 展示形式，默认NORMA | [mode](#mode) |
| key_mode | 0表示只支持左右键改动值，1表示只支持上下键改动值 | int |


### mode
| 属性值 | 描述 | 效果 |
|:-----|:----|:----|
| NORMA| 从左往右滑动 | |
| SYMMETRICA| 从右往左滑动 | |
| RANGE | 左右两端均可滑动 | ![图片](../../image/1f90e4d7-9f0c-4385-958a-ca8880818e01.png) |
| scroll_circular | | |


## 样式说明
**只能定义在样式中。**
| 样式名 | 描述 | 值类别 |
|:-----|:----|:----|
radius  |    圆角（越大越圆）  | px  
bg_opa  |    滑动条背景透明度  | % 
size  |    滑动条大小  | % 
bg_color |  背景颜色，可设置不同part  | 色值
background_color | 同上，可设置不同part | 色值
width     |  宽度  | 像素 
heigth     |  高度  | 像素 |

## 常用事件
| 事件名 | 描述 |
|:-----|:----|
| value_changed | 值改变时持续触发 |
| released | 触屏滑动释放时触发| 

* 因为slider无法增加热区，可通过事件冒泡，在父容器上处理热区和事件。
* 可通过增加slider高度来获取更好的滑动响应。
* 可在value_changed处理逻辑实现实时效果，若考虑性能则建议在value_changed时保存值，released时处理实际逻辑。
* 如果只有原生导航，没有触屏，slider值的初始化和改变可通过绑定不同的data变量来解决闪和卡顿问题。

```
<div class="container">
    <slider class="slider" :value="volume_init" max_value="10" @value_changed="changeCb()"></slider>
</div>

# 初始化
this.data.volume_init = Math.floor(this.deviceSettingInfo.speaker_value / 10);
this.data.volume = Math.floor(this.deviceSettingInfo.speaker_value / 10);

# 改变
changeVolume(val) {
    console.log("[setting_volume] 改变音量", Number(val));
    this.data.volume = Number(val);
    this.setVolume(Number(val) * 10);
},
setVolume(volume) {
    if (volume < 0) {
        volume = 0;
    }
    if (volume > 100) {
        volume = 100;
    }
    this.device.setVolume(parseInt(31 / 100 * volume));
}
```
