* 常用于画直线或折线

## 快速开始
![图片](../../image/0812f6c6-d3de-440d-a858-3a47ac8a0ecf.png)

```
<div class="container"  >
     <line class="line_style" points=[{"x":10,"y":10},{"x":50,"y":50},{"x":100,"y":10}] /> 
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.line_style {
    line_width: 10; // 线条粗度
    line_rounded: false; // 转折处是否圆滑
    line_color: #00d5ff; // 线条颜色
}
```

## 常用属性
**只能定义在dom上。**
| 属性名 | 描述 | 值类别 |
| :-------- | :-------- | :-------- |
| points | 可通过json输入多个点，将按顺序连接 | json |

## 常用样式
**只能定义在样式中。**
| 样式名 | 描述 | 值类别 | 效果 |
| :-------- | :-------- | :-------- |:-------- |
| line_width   | 线条粗度             | 像素     |
| line_rounded | 转折处是否使用圆角,true圆滑,false不圆滑 | bool   | ![图片](../../image/0efaa52d-9d76-4d20-8715-7b6305b14ae4.png) ![图片](../../image/0812f6c6-d3de-440d-a858-3a47ac8a0ecf.png) |
| line_color   | 线条颜色             | 色值   |
