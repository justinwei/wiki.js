## 快速开始
* 用于显示普通文本
![图片](../../image/d3b3e844-cccd-4a7f-a3ee-b7067e13b576.png)

```
<div class="container">
    <label class="label" text="我是测试文本的内容" align="center"></label>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.label {
    font: simsun18;   // 字体
    color: #000000; // 文字颜色
}
```

* 用于显示lvgl官方图标
![图片](../../image/0c4f4544-b7b8-4810-8eba-d5e47146bd0d.png)


```
<div class="container">
    <label class="symbol" symbol="59116"></label>
    <label class="symbol" symbol="59116"></label>
    <label class="symbol" symbol="59116"></label>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;

    display: flex;           // 弹性布局
    flex-flow: row nowrap;
    justify-content: center;
    align-content: center;
    pad_column: 12;          // 水平间距
}

.symbol {
    font: font_icon;  // 字体
    color: #000000; // 文字颜色
}
```

## 常用属性
**只能定义在dom上。**
| 属性名  | 描述 | 值类别 | 例子|
| :--------- | | :--------- | | :--------- | | :--------- | 
| text      | 文本内容 | string | `<label text="你好" />`           |
| recolor    | 是否允许改变字体颜色(可根据用户交互动态改变字体颜色)   | bool           | `<label recolor="true" />`    |
| symbol    | 导入小图标至label中,其中unicode值在下面,可自己导入图标 | unicode:int *6 | `<label symbol="61441" />`       |
| ins_txt   | text结尾处插入文本                                           | string         | `<label ins_txt="abc" />`     |
| long_mode | 超长处理方式 | string         | `<label long_mode="属性值" />`                              |

### text
* 通过data绑定数据，可使用\n换行

```
<div class="container">
    <label class="label" :text="content" align="center"></label>
</div>

data: {
    content:"我是测试文本的内容，而且内容会非常的长，\n非常的长，非常的长啊"
},
```

### long_mode
* 需要给定label的width或height属性后，label容器无法适配文本大小后long_mode属性才生效。
* 不给定label的width和height，label的高宽根据文本内容自适应，如果超出屏幕，则产生滚动条。
​
| 属性值| 描述 | 效果 |
| :--------------- | :--------------- | :--------------- |
| wrap            | 根据宽度自动换行并剪切超出部分，默认  | ![图片](../../image/f0b89838-2750-41e7-85ce-0d6340be4580.png) |
| clip              | 不换行，剪切掉超出部分 | ![图片](../../image/60f0efde-6538-4631-b97c-1cc3863fbed7.png) |
| dot              |  超出部分使用...省略号 | ![图片](../../image/cf1732d0-5785-4921-9c91-b5d129b58b90.png) |
| scroll           | 水平来回滚动或垂直来回滚动，水平滚动的优先级更高 | ![图片](../../image/bee469da-c0ad-4b61-af5b-71fc7345d43e.png) |
| scroll_circular | 回环,跑马灯循环效果 |  ![图片](../../image/4b36a90a-ac4d-4c52-a26d-d3c357da74d5.png) |

```
<div class="container">
    <label class="label" text="我是测试文本的内容，而且内容会非常的长，非常的长，非常的长啊" align="center" long_mode="clip"></label>
</div>

.label {
    width: 60;
    height: 100;
    font: simsun18; // 字体
    color: #000000; // 文字颜色
}
```

##  常用样式
**只能定义在样式中。**
| 样式名 | 描述 | 值类别 |
| :--------------- | :--------------- | :--------------- |
| width  | label宽度,不指定则auto | 像素 |
| heght | label高度,不指定则auto | 像素 |
| color  | 文本或图标的颜色  | 色值 |
| font    | 文本或图标的字体 | string |
