## 编译库文件的方式
一、设备端
1. 下载全志linux版本的交叉编译工具
	wget https://launchpad.net/gcc-arm-embedded/4.9/4.9-2015-q2-update/+download/gcc-arm-none-eabi-4_9-2015q2-20150609-linux.tar.bz2  
2. 解压：
	tar -xf gcc-arm-none-eabi-4_9-2015q2-20150609-linux.tar.bz2
3. 编译器路径：
	请放到如下路径 /usr/local/user_compuler/compiler
4. 在工程根目录下执行
    ①. 配置裁剪库
 ```
    make menuconfig
```
    ②. 编译库
```
    make   
```
5. 项目路径生成build文件夹
6. 把文件夹下面的include拷贝到设备端sdk中project/solution/Gulite_SF/目录下
7. 把文件夹中的libgulitesf.a拷贝到sdk lib目录下即可

二、模拟器
请更新模拟器工程
模拟器修改makefile：
DIRS变量修改为如下
```
DIRS 				:= $(shell find $(SRC_DIR) -type d | grep -v $(SRC_DIR)gulite_app_engine/rtos_depend \
							| grep -v $(SRC_DIR)gulite_app_engine/build  \
							| grep -v $(SRC_DIR)gulite_app_engine/si_Gluite_SF\
							| grep -v $(SRC_DIR)gulite_app_engine/scripts/kconfig)
```

INC变量增加如下内容：
```
INC 				+= -I./gulite_app_engine/Gulite_SF/components/network_protocols/mqtt/MQTTClient-C/src/
INC 				+= -I./gulite_app_engine/Gulite_SF/components/network_protocols/mqtt/MQTTClient-C/src/gulitesf/
INC 				+= -I./gulite_app_engine/Gulite_SF/components/network_protocols/mqtt/MQTTPacket/src/
```
