## key事件说明
引擎会自动加一个参数keycode的参数到事件回调
keycode值是键值的ascii码，控制键列表如下：
```
    LV_KEY_UP        = 17,  /*0x11*/
    LV_KEY_DOWN      = 18,  /*0x12*/
    LV_KEY_RIGHT     = 19,  /*0x13*/
    LV_KEY_LEFT      = 20,  /*0x14*/
    LV_KEY_ESC       = 27,  /*0x1B*/
    LV_KEY_DEL       = 127, /*0x7F*/
    LV_KEY_BACKSPACE = 8,   /*0x08*/
    LV_KEY_ENTER     = 10,  /*0x0A, '\n'*/
    LV_KEY_NEXT      = 9,   /*0x09, '\t'*/
    LV_KEY_PREV      = 11,  /*0x0B, '*/
    LV_KEY_HOME      = 2,   /*0x02, STX*/
    LV_KEY_END       = 3,   /*0x03, ETX*/
```
代码例子：


```
...... layout.xml

<button @key=key_cb("abc") >
......script.js

key_cb:fuction(name, key_code){
    console.log("name=", key, "key_code=", key_code）
}

如果按了 1键
返回
name=abc key_code=49
```

## GESTURE事件说明
引擎会自动加一个参数gesture_dir,(手势方向）的参数到事件回调
gesture_dir的值列表：
    LV_DIR_NONE     = 0x00,
    LV_DIR_LEFT     =  1,
    LV_DIR_RIGHT    = 2
    LV_DIR_TOP      =4,
    LV_DIR_BOTTOM   = 8,
    LV_DIR_HOR      = 3,
    LV_DIR_VER      = 12,
    LV_DIR_ALL      = 15,
