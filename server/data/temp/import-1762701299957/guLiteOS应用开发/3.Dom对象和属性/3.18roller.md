![图片](../../image/50b54d16-c087-435f-9b6b-5b06f9424878.png)

```
<roller class='roller_silence' align='center' > </roller>
.roller_silence{
    background-color: #e2221b;
    border_width: 0;
    outline_width: 0;
   }
```

## 属性说明
**只能定义在dom上。**
| 属性名称 | 描述 | 值类别 | 例子 | 
|:-----|:----|:----|----|:----|
| options | 选项内容 | string |```<roller options="1" /> ``` |
| selected | 选择起始行数（第一行为第零行） | int |```<roller  :selected="0" /> ``` |
| visible_row_count | 可见行数 | int |```<roller visible_row_count="3" /> ``` |
| alian | 相对定位 | alian |```<gif :alian='center' /> ``` |
| x | 定位，dom的x坐标 | px |```<gif x="50" /> ``` |
| y | 定位，dom的x坐标 | px |```<gif y="50" /> ``` |
| pos | 定位，相对定位 | px |```<gif pos="50 50" /> ``` |
| size | dom的大小，设置width和heigth | px |```<gif size="100 100" /> ``` |
| show | dom是否显示 | bool |```<gif :show="true" /> ``` |
## 样式说明
**只能定义在样式中。**
| 样式名称 | 描述 | 值类别 | 例子 | 
|:-----|:----|:----|----|:----|
|background-color | 背景颜色 | 色值 |```background-color: #b0cc13; ``` |
| width | 宽度 | px |```width: 400px;``` |
|radius| 圆角|px|```radius: 50px;```|

## 常用事件
| 事件名称 | 描述 | 例子 | 
|:-----|:----|:----|----|:----|
|click |点击| ```@click='fn()'``` |
|value_changed |值改变| ```@value_changed='fn()'``` |
例子

```
<roller class='roller_silence' align='center' :options='options'@value_changed='cd()' > </roller>
module.exports = {
    data: {
        options: "1\n2\n3\n4\n5\n6\n7\n8\n9\n10\n11\n12\n13\n14\n15\n16\n17"
    }
}
```
