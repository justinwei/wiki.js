* 用于呈现进度，只能通过改变value来改变进度。
* 不同于slider，bar在触屏和原生按键导航下均不能拖动，

## 快速开始
![图片](../../image/bcfdc36b-f228-4738-875e-cfd2f645e699.png)

```
 <div class="container">
    <bar class="bar" value="30" max_value="100" align="center"/>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.bar {
    height: 20;
    width: 200;
    background-color: #0080FF; //淡蓝,背景条颜色
    radius: 20; // 圆角

    &:PART_INDICATOR {
        background-color: #3300FF; //深蓝,进度条颜色
    }
}
```

## 常用属性
**只能定义在dom上。**
| 属性名  | 描述 | 值类别 |
| :--------- | :--------- | :--------- |
| value    |  当前值    | int |
| max_value |  允许的最大值    | int | 

## 常用样式
**只能定义在样式中。**
| 样式名  | 描述 | 值类别 |
| :--------- | :--------- | :--------- |
| width    |  宽度    | 像素 |
| height  |  高度    | 像素 | 
| radius | 进度条的圆角,可设置最大值自适应圆角 | 像素 |
| bg_color | 背景色，可设置不同的part | 色值 |
| background-color |  同bg_color | 色值 |

### width、height说明
* 如果高度>宽度，将创建一个竖直方向的进度条。
![图片](../../image/e1e6e7e5-49f2-4043-b531-23e9eaeba059.png)

## 常用事件
无
* 一般通过value绑定data属性来持续改变进度值
