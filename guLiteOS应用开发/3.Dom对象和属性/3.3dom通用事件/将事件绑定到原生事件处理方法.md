**目的**
为了可以快速响应事件处理，可以将事件响应使用C语言原生开发。
例如slide控件，监听value_change事件时，太多事件发到JS，会导致JS队列快速排满，CPU爆满，界面卡死

以下使用原生处理slide change事件的例子来说明
**C原生开发**

1. 开发事件处理函数
```
void change_volume_handle(lv_event_t *event)
{
  lv_obj_t *lv_obj =  event->current_target;
  if(lv_obj == NULL){
    printf("system_event_change_volume lv_obj is null\n");
    return;
  }
  int value = lv_slider_get_value(lv_obj);
  printf("system_event_change_volume %d\n", value);
}
```

2. 将处理函数注册到系统中
 _**所有的注册原生的名称必须以 system_event 开头**_ 
```
    guAddAppEventHandle("system_event_change_volume", change_volume_handle);
    guAddAppEventHandle("system_event_change_bright", change_bright_handle);
```

**JS APP开发**
包括以下两类
1. layout.xml绑定事件
```
        <slider  width="200" mode="RANGE"  @value_changed="system_event_change_volume()" />        
```

2. 通过动态创建的函数
```
        var tag_0 = gu_ui.createDom(null, "slider")
        gu_ui.setDomAttr(tag_0, "width", "100")
        gu_ui.setDomAttr(tag_0, "max_value", "100")
        gu_ui.setDomStyle(tag_0, "height", "20", '')
        gu_ui.setDomEvent(tag_0, "value_changed", "system_event_change_volume")
```
