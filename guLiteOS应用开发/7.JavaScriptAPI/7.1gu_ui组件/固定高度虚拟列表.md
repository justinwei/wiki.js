**目标：**
解决一个列表同时滚动上千条数据，
（本方案技术来源于 https://juejin.cn/post/6844903982742110216）

div 增加5个属性
virtual_list_data，虚拟列表绑定的路径，必须和子结点的v-for保持一致
virtual_list_position，初始位置，是读写属性，位置发生变化时，这个属性会自动改变，
virtual_list_focused_item_data，选中后的样式属性绑定值
virtual_list_focused_index，指定焦点的索引
virtual_list_set_focused，设置虚拟列表的焦点（一个页面多个虚拟列表时需要设置好）

例子如下:
layout.xml
```
    <div class="space" width="240" :height="height" virtual_list_data="all_list" :virtual_list_set_focused="show" :virtual_list_position="position" :virtual_list_focused_item_data="focus_item" :virtual_list_focused_focused_index="focused_index">       
    <!-- class 需要设置背景颜色 width必须固定，is_virtual_list=true，height=以下v-for节点高度*单页显示个数 -->
        <div class="btn_container_col" scrollbar_mode="MODE_OFF" height="auto" width="240" gridnav="rollover">
            <!-- class必须设置为collum的flex布局 pad_row=0,  height为"auto" width需要和父节点一样 -->
            <div class="btn" height="46" width="240" focus_key="item.focus" scrollbar_mode="MODE_OFF" v-for="item in all_list"  @click="click_me(item.id)">
                <!-- height高度必须为固定高度，单项的高度 = 虚拟列表高度 / 一屏显示个数  -->
                <img src="file:assets_images_icon_collect_f.png" class="item_img" />
                <label class="sub_lbl" width="240" :long_mode="item.long_mode" :text="item.name"   />
            </div>
        </div>   
    </div>         
```
css
```
.space{
    background:#ffffff;    
}
.btn_container_col{
    height: 20;        
    background:#ffffff;
    display: flex;    
    flex-flow: column nowrap;  
    align-items:center;   
    pad_row:0;       
}
.btn:focused|focus_key {
    background: #00ff00;
}
.btn {
    background:#FFFFFF;
}
```
