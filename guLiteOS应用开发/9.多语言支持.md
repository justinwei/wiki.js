## guLiteOS支持多语言资源文件配置，支持多语言切换

### 1. 资源文件 resouce.json
资源文件放在APP的根目录下，格式如下
```
{  
    "en": {
        "message": {
            "hello": "hello world"    
        }  
    }, 
    "cn":{
        "message":{
            "hello":"你好，世界"
        }
    }
}

```

默认语言是cn
### 2. 在layout.xml使用资源

```
    <label class="text" text="$t('message.hello')" />

默认语言是cn时等同于：
   <label class="text" text="你好，世界" />
```

### 3. 在data.json使用资源
```
{
    "title_st":"$t('message.hello')"
}

默认语言是cn时等同于：

{
    "title_st":"你好，世界"
}
```
### 4. 在javascript使用资源

```
     console.log(gu_ui.get_resource_value("message.hello"))
输出结果：
    你好，世界

```


### 5. 语言切换
```
     gu_ui.set_locale('en');
     console.log(gu_ui.get_resource_value("message.hello"))
输出结果：
    hello world

```
