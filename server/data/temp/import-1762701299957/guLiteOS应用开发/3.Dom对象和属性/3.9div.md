## 用于原生导航
gridnav
https://docs.lvgl.io/master/others/gridnav.html#usage

gridnav_flag 属性
| 属性值 | 描述 | lv对应值 |
|:-----|:----:|:----|
| NONE|  |LV_GRIDNAV_CTRL_NONE|
| ROLLOVER|  |LV_GRIDNAV_CTRL_ROLLOVER|
|SCROLL_FIRST |  |LV_GRIDNAV_CTRL_SCROLL_FIRST|
用于键盘导航 button或text 等可以获得焦点的dom对象

```
..... layout.xml

<div gridnav="ROLLOVER">
     <button >
        <label txt="1" />
     </button> 

     <button >
        <label txt="2" />
     </button> 

     <button >
        <label txt="3" />
     </button> 

</div>

```

设置了 gridnav=true 的div 子/孙dom 不能再使用 div，否则子div下的dom无法获得焦点


```
..... layout.xml
<div gridnav="ROLLOVER">
    <div >
         <button >
            <label txt="1" />
         </button> 
    
         <button >
            <label txt="2" />
         </button> 
    
         <button >
            <label txt="3" />
         </button> 
    </div>
</div>

//这样按钮将不会获得焦点
```
**注意：循环滚动和gridnav不能并存，一个页面不能同时有gridnav和circular_scroll=ture**

## 循环滚动样例，横向滚动 1个大图标
layout.xml
```
    <!-- 单个大图标的菜单 -->
    <div :show="app_list_show" id="app-list" class="apps" circular_scroll="true"   scroll_anim_time="400" scroll_snap_x="3">
        <button v-for="app in appList" class="app_btn"  @click="onClickApp(app.id)">
            <img :src="app.src" align="center"/>
        </button>
    </div>
```

style.css

## 循环滚动样例，垂直滚动，仿roller
![图片](../../image/c6a9627c-54e2-4f55-aa05-533c7811e8e2.png)
layout.xml
```
    <!-- roller快速滚动 -->
    <div :show="column_list_show" class="column_list_bg" align="center" scrollbar_mode="MODE_OFF">
        <!-- roller选中框 -->
        <div class="column_list_selected" align="center" scrollbar_mode="MODE_OFF"> </div>
        <!-- roller内容 -->
        <div class="column_list_container" circular_scroll="true"  align="center"  scroll_anim_time="300" scroll_snap_y="3" scrollbar_mode="MODE_OFF" >
            <div v-for="item in column_list" class="column_list_item">        
                <label  :text="item.id" align="center"/>
            </div>
        </div>    
    </div>
```
