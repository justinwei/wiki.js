## 快速开始
![图片](../../image/102f34eb-7863-4da8-810b-5eb7608c2764.png) ![图片](../../image/1d23ce23-cb99-48ef-bd20-b03eff0635bc.png) 

```
<div class="container">
    <switch value="true" class="switch" align="center" @value_changed="changeCb()" @click="clickCb()"></switch>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.switch {
    height: 20;
    width: 40;
    bg_color: #BEBEBE;
    radius: 22;
    bg_opa: 255;

    &:PART_INDICATOR|CHECKED {
        bg_color: #00FF00;
    }
}

clickCb() {
        logger.log("点击了switch");
},
changeCb(val) {
        // 1打开0关闭
        logger.log("switch状态改变,val=", val);
},
```

## 常用属性
只能定义在dom上。
|属性名 | 描述 | 值类型|
|:----------- | :----------- | :-----------|
| value  |  是否开启，true开启 | bool

## 常用样式
只能定义在样式中。
|属性名 | 描述 | 值类型|
|:----------- | :----------- | :-----------|
| radius| 圆角 | 像素 |
| bg_color | 背景色,可设置part | 色值 |

## 常用事件
|事件名 | 描述 |
|:----------- | :----------- |
| value_changed  |  状态改变时触发 | 
| click | 点击事件 |

* 因为switch无法增加热区，可通过在switch上增加事件冒泡，在父容器上处理热区和事件。
