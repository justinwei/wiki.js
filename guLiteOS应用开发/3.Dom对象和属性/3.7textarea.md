* 文本域相当于Label + 光标（cursor），但是有更丰富的字符处理能力，比如过滤，动态增减等。
* 常用于：WIFI密码输入、电话号码输入等动态场景。

## 快速开始
![图片](../../image/6088a159-8bae-44f4-9b88-ca5299b496d7.png)

```
<div class="container">
    <textarea :text="content" placeholder_text="请输入密码" :add_text="addChar" :del_char="delChar" focus="true" class="textarea" align_pos="top_mid 0 50"></textarea>
    <button size="50 20" @click="clickAdd()" align_pos="top_mid -30 100">
        <label text="add" align="center"></label>
    </button>
    <button size="50 20" @click="clickDel()" align_pos="top_mid 30 100">
        <label text="del" align="center"></label>
    </button>
</div>

data: {
        content: "",
        addChar: "",
        delChar: false,
},
clickAdd() {
    this.data.addChar = "a"; //每次点击增加a字符
},
clickDel() {
    this.data.delChar = true; //每次点击删除一个字符
},

.container {
    height: 100%;
    width: 100%;
    background-color: #000000;
    radius: 0;
    bg_opa: 255;
}

.textarea {
    width: 150;
    height: 30;
    background-color: #FFFFFF;
    radius: 0;

    font: simsun18;
    color: #000000;
}
```

## 常用属性
**只能定义在dom上。**
| 属性名 | 描述 | 值类别 |
| :---------| :---------| :---------|
| text | 绑定data可给文本域默认值或获取到当前值 | string |
| one_line | 文本框设置为单行输入模式 | bool |
| password_mode | 是否启用密码模式，启用则显示输入字符后将替换成星号 | bool |
| accepted_chars | 只接收指定的字符串中的字符  | string |
| max_length | 文本域可容纳的最大字符数 | int |
| add_text | 在光标位置插入文本 | string |
| placeholder_text | 当文本域无内容时，设定占位文本 | string |
| del_char | 是否删除光标左侧的一个字符，每触发一次删除一个 | bool |
| del_char_forward | 是否删除光标右侧一个字符，每触发一次删除一个 | bool   |
| cursor_pos | 移动光标到特定位置，搭配其他属性使用，跳过的字符数 | int  |
| focus | 是否启用闪烁的光标 | bool | 
| text_selection | 是否启用文本选择功能，支持选择文本的任何部分 | bool | 

### one_line
* 启用单行模式后，文本域高度自动设置为仅显示一行，不自动换行，将产生横向滚动条。

### accepted_chars
* 当text和add_text属性不绑定data，而是给默认值时，accepted_chars必须定义在text和add_text属性前，才触发过滤。绑定data，默认赋值受也有此问题，动态赋值不受影响。
* accepted_chars和max_length属性公用也有此问题。

## 常用样式
**只能定义在样式中。**
| 样式名 | 描述 | 值类别 |
| :--------- | :---------| :---------
| width | 文本域宽度 | 像素 |
| height  | 文本域高度 | 像素 |
| font | 字体 | string |
| color | 字体颜色 | 色值   |
| background-color | 文本域背景颜色 | 色值  |
| bg_color | 同上，文本域背景颜色 | 色值  |
| radius | 外边框圆角 | int |

### width、height
* 如果指定了高宽，文本内容超过textarea的宽度，会自动换行，此时如果高度再超过textarea的高度时会出现纵向滚动条。
* 如果宽度auto，不会自动换行，但是会产生横向滚动条。
