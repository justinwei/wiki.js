# 2. 多APP开发说明
# 2.1 APP状态
APP当前支持以下三种状态：
* 运行状态：
* 挂起状态：页面和图片的缓存保留，只是退出所有页面
* 退出状态：所有缓存都将释放

# 2. 2 APP跳转说明
 * 运行状态 ->  挂起状态： suspendApp()
  * 挂起状态 ->  运行状态： activeApp()
 *   运行状态调转：jumpApp()

* 例1:挂起当前APP，进入另一个APP
```
     gu_ui.suspendApp();
     gu_ui.jumpApp("xmly"); //进入另外的APP
```

* 例2:退出当前APP，激活另一个已经挂起的APP
```
        gu_ui.exitApp();  // 退出当前APP
        gu_ui.activeApp("launch") //激活另外一个APP
        gu_ui.jumpPage("index", "") //进入指定激活页面APP的页面
```
* 例3:退出当前APP，跳入另外的APP
```
        gu_ui.exitApp();  // 退出当前APP
        gu_ui.jumpApp("xmly"); //进入另外的APP
```




# 2. 3 APP的js代码
  和页面一样，所有的变量和函数要放在APP名称命名的变量里，例如：
```
//baidupan是APP的名称，当运行jumApp函数时，会自动运行baidupan.onload()
baidupan = {
    onload:function(){
        console.log("enter baidu pan app");
        gu_ui.jumpPage("index", "")
    }
}

```
