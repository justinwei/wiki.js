* 选项卡常用呈现左右滑动、上下滑动等多栏目效果。

## 快速开始
![图片](../../image/fe484e28-bab8-4898-b700-d62ca3c42173.png)

```
<div class="container">
    <tabview class="tabview" tab_pos="DIR_TOP" tab_size="30" act="0" scroll_anim="true" scroll_anim_time="1000" @value_changed="changeCb()">
        <tab name="tb1">
            <label text="tab 1" />
        </tab>
        <tab name="tb2">
            <label text="tab 2" />
        </tab>
        <tab name="tb3">
            <label text="tab 3" />
        </tab>
    </tabview>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.tabview {
    bg_color: #00BFFF;
}

changeCb(val) {
    logger.log('changeCb,val=', val);
},
```

## 常用属性
**只能定义在dom上。**

| 属性名 | 描述 | 值类别 |
:----------- | :----------- | :----------- |
tab_pos | 设置选项卡栏位置 | DIR_LEFT/RIGHT/BUTTON/TOP  |
tab_size | 设置选项卡栏大小,水平排列则是宽度值,垂直排列则是高度值 | 像素 |
act | 选中的tab,0表示第一个tab | int |
show_tab_btns | 是否显示选项卡栏,默认true | bool |
scroll_anim |  改变act时，是否有动画效果,默认false   | bool |
scroll_anim_time | 动画效果滚动速度,越小越快 | 毫秒 |
disable_scroll |  是否禁止手势滚动tabview  | bool |

### tab_pos
属性值| 描述 |
:----------- | :----------- |
DIR_LEFT | 选项卡栏在左侧 |
DIR_RIGHT | 选项卡栏在右侧 |
DIR_TOP | 选项卡栏在顶部 |
DIR_BOTTOM | 选项卡栏在底部 |
DIR_NONE | |
DIR_HOR | | 
DIR_VER | |
DIR_ALL | |

### disable_scroll
* 当需要获得手势效果时，此属性设置为true，且tab配合clear_flag="GESTURE_BUBBLE" @GESTURE="gestureCb()"使用，不会触发tabview上的value_changed事件。

## 常用样式
**只能定义在样式中。**
样式名 | 描述 | 值类别 |
:----------- | :----------- | :----------- |
background-color | 背景色 | 色值 |

## 常用事件
事件名 | 描述 |
:----------- | :-----------  |
value_changed | 值改变事件,返回改变后的act值 |
click | 点击事件 |
GESTURE | 手势事件,返回手势方向,1手左滑2手右滑 |

### GESTURE
* 此事件绑定在tab上，且tabview要设置disable_scroll="true"
* 此事件需要配合clear_flag="GESTURE_BUBBLE" 使用才生效。

```
<div class="container">
    <tabview class="tabview" tab_pos="DIR_TOP" tab_size="30" :act="tabIndex" disable_scroll="true">
        <tab v-for="item in tabs" :name="item.name" clear_flag="GESTURE_BUBBLE" @GESTURE="gestureCb()">
            <label :text="item.content" />
        </tab>
    </tabview>
</div>

data: {
    tabIndex: 0,
    tabs: [
        { name: "tb1", content: "tab1" },
        { name: "tb2", content: "tab2" },
        { name: "tb3", content: "tab3" }
    ]
},
gestureCb(val) {
    logger.log('gestureCb,val=', val,typeof val);
        if (val === "1") {
            if (this.data.tabIndex > 2) {
                return;
            }
            this.data.tabIndex++;
        }
        if (val === "2") {
            if (this.data.tabIndex < 0) {
                return;
            }
            this.data.tabIndex--;
        }
},
```
