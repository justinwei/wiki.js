#####  1. 安装**SDL**
>  官网地址: https://www.libsdl.org/

-  Linux
```
    sudo apt-get update && sudo apt-get install -y build-essential libsdl2-dev
```

- MacOS(M1 &M2)
> 源码地址： https://github.com/libsdl-org/SDL/releases/

```
  解压SDL在其根目录依次执行:
  ./configure
  make
  make install 
```

#####  2. 安装**mxml**
>  源码地址: https://github.com/michaelrsweet/mxml/releases/tag/v3.3.1

- Linux
```
 configure 
 make
 make install
```
 
- MacOS(M1 &M2)
```
  intel ./configure
  intel make
  intel make install 
```

####  3. 安装ffmpeg

必须选择 v6.1.2版本
```

####  4. 安装压缩工具-**xz**

- MacOS
```
