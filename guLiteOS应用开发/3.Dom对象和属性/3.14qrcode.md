## 快速开始
![图片](../../image/b6fcc187-ffbc-4f94-a6ed-705f99cf0b16.png)

```
<div class="container">
    <qrcode qr_size="100" bg_color="000000" fg_color="FFFFFF" data="www.baidu.com" align="center" class="qrcode"></qrcode>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.qrcode {
    border_width: 1px;
    border_color: #000000;
}
```

## 常用属性
**只能定义在dom上。**

| 属性名称 | 描述 | 值类别 |
|:-----|:----|:----|----|:----|
| bg_color | 背景颜色，默认是白色 | string 色值 |
| fg_color | 前景颜色，默认是黑色 | string 色值 |
| qr_size | 二维码大小  | int 像素 |
| data | 二维码内容  | string |

## 常用样式
**只能定义在样式中。**

|样式名 | 描述 | 值类型|
|:----------- | :----------- | :-----------|
| border_width | 外边框宽度    | 像素|
| border_color | 外边框颜色 | 色值 |
