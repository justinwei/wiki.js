## 快速开始
![图片](../../image/108fd94d-68c4-4975-b580-759b7747b458.png)

```
<div class="container">
 <arc  center="true" size="100 100" rotation="135" bg_start_angle="0" bg_end_angle="360" remove_style="part_KNOB"/>
</div>
.container {
    .g_container(100%, 100%, #fdfdfd, 0, 255);
}

```

## 常用属性
**只能定义在dom上。**

属性 | 描述 | 值类别|例子
:----------- | :-----------: | -----------:| -----------:
rotation      |      旋转  |       int     |      `<arc rotation="2"/>` 
value      |      初始值大小 |       int     |     `<arc value="50"/>`
bg_angles      |       背景初始，结束角度     |     int | `<bg_angles ="0 360"/>`
bg_start_angle    |      背景初始角度 |        int      |    <bg_start_angle="0"/>`` 
bg_end_angle      |      背景结束角度  |       int     |     `<bg_end_angle="360"/>`
angles       |         前景初始，结束角度     |   int  |  `<angles ="0 180"/>`  
start_angle      |      前景初始角度  |       int     |     `<start_angle="0"/>`  
end_angle    |      前景结束角度  |       int     |   `<end_angle="180"/>`
range   |      设置最大值，最小值  |       int     |   `<rang ="0 200"/>` 
change_rate   |      变化率(越大变化越快) |       int     |  `<change_rate ="500"/>` 
mode   |      模式 |      NORMAL/REVERSE/SYMMETRICAL    |  `<mode='NORMAL'/>` 
size  |      大小 |      px |  `<size='100 100'/>` 

### mode

模式| 描述 
:----------- | :-----------: |  :-----------: | 
 NORMAL      |     普通模式。指示器从最小值绘制到当前值    |    ![图片](../../image/30a95446-c55e-47ea-8ba0-32964d30d791.png) |
 REVERSE    |     反向模式。指示器从最大值绘制到当前值逆时针绘制    |   ![图片](../../image/509cc5b4-7f31-48e0-8081-606d328ac576.png) |
 SYMMETRICAL     |     对称模式。指示器从中间点（中间值）绘制到当前值    |   ![图片](../../image/7681e27b-6dc3-4a3b-8489-8483b7610e46.png)|

### 常用样式
**只能定义在样式中。**

属性 | 描述 | 值类别|例子
:----------- | :-----------: | -----------:| -----------:
width      |      宽度  |      px   |      `width:10` 
heigth     |     高度  |     px    |      `heigth:10` 
arc_color     |     颜色  |     色号    |      `arc_color:#FF00FF` 
arc_width    |     圈的宽度  |     int    |      ` arc_width: 10;` 
## 例子
![图片](../../image/a9bd468c-91cb-4d1b-b69c-de4379905783.png)

```
<div class="container">
 <arc class='spinner'  center="true" size="100 100" angles="0 180" bg_start_angle="0" bg_end_angle="360"  mode='NORMAL' value='50' range='0 200' change_rate='100' />
</div>

.container {
    .g_container(100%, 100%, #fdfdfd, 0, 255);
}

.spinner {
    arc_width: 10; //设置内圈大小
    arc_color: #FF00FF;//内圈颜色
    &:PART_INDICATOR {
        arc_color: #FFFF00;//外圈颜色
    }
}
```
