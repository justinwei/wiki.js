### 一、模块介绍
设备信息管理模块
### 二、api
#### 1、device.setVolume(value)
设置音量值
* value: 整数，音量值
  
#### 2、device.getVolume()
获取音量值
* 返回值：返回当前音量大小

#### 3、device.setBrightness(value)
设置亮度
* value: 整数，亮度值

#### 4、device.getBrightness()
获取当前亮度
* 返回值：返回当前亮度
  
#### 5、device.getMac()
获取mac地址
* 返回值：字符串，mac地址

#### 6、device.getChipId()
获取chipId
* 返回值：字符串，chipId

#### 7、device.poweroff()
关机，无参数，无返回值
  
#### 8、device.reboot()
重启，无参数，无返回值

#### 9、device.calRtc([callback])
异步校准rtc
* callback: 可选参数，function(err), err为0表示校准成功，-1则为校准失败

#### 10、device.calRtcSync()
同步校准rtc
* 返回值: 0表示校准成功，-1则为校准失败

#### 11、device.setDispRotation(angle)
设置屏幕旋转角度
* angle：整数，旋转角度, 90度为一个旋转单位

#### 12、device.earphoneInit()
初始化耳机驱动,  无参数，无返回值

#### 13、device.getEarphoneState()
获取耳机状态
* 返回值: 1表示插入，0为未插入

#### 14、device.earphoneDeinit()
反初始化耳机驱动,  无参数，无返回值

#### 15、device.getTimeMs()
获取系统运行时间
* 返回值: 系统运行的毫秒数

### 三、例子
```
var device = require('device');

device.setVolume(20);

console.log("volume = " + device.getVolume());

device.setBrightness(20);

console.log("Brightness = " + device.getBrightness());
console.log("mac = " + device.getMac());
console.log("chipid = " + device.getChipId());

device.poweroff();
device.reboot();

device.setDispRotation(90);

device.calRtc(function(err) {
    console.log(err);
})

device.earphoneInit();
var ret = device.getEarphoneState(); if (ret == 1) {console.log('耳机已插入');}
device.earphoneDeinit();

var ret = device.getTimeMs();
console.log('getTimeMs = ', ret);    
```
