* 1个dom并非支持以下所有属性。

|:-----|:-----|:-----|

| x | 定位，dom的x坐标 | units |
| y | 定位，dom的y坐标 | units |
| align | 定位，相对定位 | align |
| align_to | 定位，相对id定位再加偏移  | align |
| align_pos | 定位，相对定位 |align |
| pos | 定位，绝对定位，同时设置x和y坐标，空格分隔 | size  |
| size | dom的大小，分别设置width和height，空格分隔 | size |
| show | dom是否显示 | bool |
| flag | 设置flag | [flag](#flag) |
| clear_flag | 清除flag | flag |
| id | dom的ID，配合align_to使用  | string |
| center | 居中 | bool |
| scrollbar_mode | 设置滚动模式  | [scrollbar_mode](#scrollbar_mode)  |
|  state| 设置状态 | [state](#state)  |
|  clear_state| 清除状态 | [state](#state)  |
|  gridnav| [按键控制焦点](https://young192.coding.net/p/guliteos/wiki/31)  |bool |
|  focus| 设置焦点 | bool |
|  focus_key| 设置key焦点 | bool |
|  group_focus| 设置group光标 |bool |
|  gridnav_focus| 设置gridnav里的元素的焦点 |bool |
|  scroll_to_view| 特定内容滚动到可视区底部 |bool |
|  scroll_anim| 设置滚动是否有动画 ，影响scroll_to_x，scroll_to_y，tab_view | bool |
|  scroll_to_x| 滚动到x位置 | 像素 |
|  scroll_to_y| 滚动到y位置 | 像素 |
|  scroll_anim_time | 设置滚动的动画时间,越短越快 | 毫秒 |
|  remove_style| 清除某个part的样式| selector |
|  ext_click_area| 设置点击区域、设置热区 | units |

|  is_virtual_list| 设置是否为虚拟列表| bool  |
|  scroll_snap_x| 滚动对齐，0, 默认，1:左/上对齐 2:右底对齐，3:居中| int  |
|  scroll_snap_y| 滚动对齐，0, 默认，1:左/上对齐 2:右底对齐，3:居中| int  |
|  index| 设置/获取 对象相对父对象的index| int  |
|  invalidate| 强制刷新对象| bool  |
|  checked| | bool  |
|  scrolled| | bool  |
