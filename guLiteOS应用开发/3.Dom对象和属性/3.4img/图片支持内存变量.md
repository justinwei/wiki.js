**目的：**
     解决图片消耗太多引擎的内存问题
     可以将图片数据放在XIP

**步骤**
1. 将图片转成C array
    https://lvgl.io/tools/imageconverter
    将转出的C代码放在SDK
   * 需要将这个代码放在XIP


2. 在SDK代码启动应用时，将图片加入列表
```
    extern lv_img_dsc_t c754f6d4c36b2479e61db1b301224c42;
    //图片名，图片地址（第一步转成C代码）
    guAddMemImg("test_img_mem", &c754f6d4c36b2479e61db1b301224c42);  
```



3. 在layout.xml里src的属性指定是内存变量
    以mem：开始，后面接图片名，在第2步设置好的。
```
     <img src="mem:test_img_mem" />   
```
