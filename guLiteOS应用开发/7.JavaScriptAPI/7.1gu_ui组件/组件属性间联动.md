**目标：**
解决一个组件的属性发生变化，自动触发其他组件的属性发生改变。
使用原生自动触发，提高刷新性能。
以下几个场景：
1. 快速滑动bar，显示百分比
2. 快速滚动首页的图标的菜单，

**方案：**
增加了一个方法设置触发器，例如：
```
this.data.trigger = gu_ui.createDomTrigger({
            trigger_dom: "app-list",  //触发的dom对象ID
            trigger_event: "focus_changed",  //触发的事件
            animation:{                                //是否支持动画，可以为空
                duration:200,                           //动画时长
                path:"ease_in"                        //动画路径，详见 gu_ui组件 动画路径，可以为空
            },
            source_dom_attr:[  //原始的dom属性值，存放变量，支持多个
                {
                    dom_id: "app-list",        //domID，
                    attr: "focus_index",        //属性值
                    valueName:"index",        //存放的变量名
                }
            ],
            dest_dom_attr:[     //触发修改的dom属性，支持多个
                {
                    dom_id: "show_lbl",   //domID，支持表达式，例如 'lbl' + list[$index].id 
                    attr: "text",                //需要修改的属性名   
                    valueName:"list[$index].id" //需要修改的值，支持表达式，这个例子的list数据从data里取。
                },
                {
                    dom_id: "'img_src_'+($index-1+7)%7",
                    attr: "zoom",
                    is_anim:true,                 //是否支持动画，动画属性只支持integer类别的属性
                    valueName:"156"
                },  
                {
                    dom_id: "'btn_'+($index+1)%7",
                    style: "bg_color",     //需要修改的样式
                    valueName:"#00000ff"
                },   
                {
                    dom_id: "dot_list",
                    attr: "focus_index",
                    valueName:"$index"
                }
            ],
            data:{list:[
                { id: '1 ai_voice'},
                { id: '2 theme'},
                { id: '3 alarm' },
                { id: '4 timer' },
                { id: '5 tomato'},
                { id: '6 stopwatch'},
                { id: '7 set' }                     
            ]}
        })     )

```

页面跳转后，或是相关的dom删除时，需要删除这个触发器
```
gu_ui.destroyDomTrigger(this.data.trigger)
```
