* 类似label，用于显示文本，可精准控制到单个字的样式
* span标签只能在spangroup标签中

## 快速开始
![图片](../../image/6c4f609b-7b02-416e-80eb-af53162bec61.png)

```
<div class="container">
    <spangroup class="spangroup" align="center">
        <span v-for="item in words" :text_color="item.color" :font="item.font" :text="item.text"/>
    </spangroup>
</div>

data: {
        words: [
            { color: "FFFFFF", font: "simsun18", text: "测" },
            { color: "00FF00", font: "simsun18", text: "试" },
            { color: "0000FF", font: "simsun18", text: "用" },
            { color: "F00F00", font: "simsun18", text: "例" },
        ]
},

.spangroup {
    height: 100;
    width: 100;
    background-color: #000000;
    radius: 0;
    bg_opa: 255;
}
```

## spangroup 常用属性
**只能定义在dom上**
| 属性名 | 描述 | 值类别 |
| :-------- | :-------- | :-------- |
| align    | 相对父容器定位,对齐方式 | 参考定位的说明 |
| overflow | 用于设置对象溢出模式 | clip、ellipsis |
| indent   | 用于设置第一行的缩进量。除了MODE_FIXED外，所有模式都支持像素单位，MODE_BREAK模式也支持百分比单位 | int    |
| mode     | 用于设置对象模式  | fixed、expand、break  |

### overflow
| 属性值 | 描述  | 效果 |
| :------- | :------- | :------- |
| clip     | 在区域的边界处截断文本(默认值)                  |    ![图片](../../image/63f0a837-bfa6-4090-8ddf-9dcb57ff84dd.png)      |
| ellipsis| 当文本溢出该区域时,显示省略号 |    ![图片](../../image/4675d927-c263-43fa-8be9-6ca90e7a2ab0.png)      |

### mode
| 属性值 | 描述  | 效果 |
| :------- | :------- | :------- |
| fixed  |开启后根据Overflow处理文本( 默认值),其他mode将会无效overflow | |
| expand  | 所有span放在同一行,自动扩展宽度 |     ![图片](../../image/538b3cd5-b6f2-4c1d-9b96-d50f4e7a7ab7.png)     |
| break  | 保持width宽度，高度根据文本大小设定   |   ![图片](../../image/b42f1a93-e544-463f-8c39-ac4631bdc710.png)       |

## span 常用属性
**只能定义在dom上**
| 属性名 | 描述 | 值类别 |
| :-------- | :-------- | :-------- |
| text     | 文本内容 | string |
| text_color | 文本颜色 |  色值 |
| font     | 文本字体 | 字体 |

## 常用样式
**只能定义在样式中**
| 样式名 | 描述 | 值类别 |
| :------ | :------ | :------ |
| width | 宽度 | 像素 |
| heigth | 高度 | 像素 |
| background-color | 背景色 | 色值  |

* 内容为连续英文字母时，该行剩余位置不能完全显示会整体换行，中文不受影响。
