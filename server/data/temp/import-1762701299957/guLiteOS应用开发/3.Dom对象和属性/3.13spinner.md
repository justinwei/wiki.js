## 快速开始
![图片](../../image/a9574550-e8b9-4716-be64-9936b6203115.png)
```
<div class="container">
    <spinner spin_time="1000" arc_length="60" size="100 100" align="center"></spinner>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}
```

## 常用属性
**只能定义在dom上。**

| 属性名 | 描述 | 值类别 |
|:-----|:----|:----|----|:----|
| spin_time | 转一圈时间，越短越快 | 毫秒 |
| arc_length | 环旋转部分长度，最大值为周长 | 像素 |

## 常用样式
**只能定义在样式中。**

| 样式名 | 描述 | 值类别 | 效果 |
|:-----|:----|:----|----|:----|
| arc_width | 环的粗细 | 像素 | ![图片](../../image/599e2a53-0685-4809-8f49-0dec192ed0a4.png) |
| arc_color | 环的背景色 | 色值 | ![图片](../../image/2893fb1e-7c3b-4666-a2f7-37f2c9e8baff.png) |
| arc_color | 旋转部分的颜色（PART_INDICATOR部分） | 色值 | ![图片](../../image/a5f0efd0-59a7-49c9-8e58-6ec1b2d06261.png) |

```
<div class="container">
    <spinner spin_time="1000" arc_length="60" size="100 100" align="center" class="spinner"></spinner>
</div>

.container {
    height: 100%;
    width: 100%;
    background-color: #FFFFFF;
    radius: 0;
    bg_opa: 255;
}

.spinner {
    arc_width: 10;
    arc_color: #FF00FF;

    &:PART_INDICATOR {
        arc_color: #FFFF00;
    }
}
```
