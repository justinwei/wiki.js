* 样式表支持 less，后台会自动将less文件转化成css文件

样式表必须放在style.css文件中，
1. dom默认样式放在 liteAPP根目录的dom.css文件
例如
```
div{
    padding-top: 0;
    padding-bottom:0;
    padding-left:0;
    padding-right:0;
    border_width:0; 
}
/* 按钮默认样式*/
button{
    padding-top: 0;
    padding-bottom:0;
    padding-left:0;
    padding-right:0;
}
/* 按钮获得焦点时的样式*/
button:focused{
    padding-top: 0;
    padding-bottom:0;
    padding-left:0;
    padding-right:0;
    bg_color:#ff0000;        
}
```

设置 dom处于某个的状态，或是dom 子部分的样式
详细参考链接（https://docs.lvgl.io/master/overview/style.html#add-and-remove-styles-to-a-widget） 关于**selector**的描述
写法用“：”分割，多个值用"|"分格：


```
...... style.css

/* 按钮获得焦点时的样式*/
btn_style:focused{
    ...
}

/* 按钮获得焦点和获得焦点时的样式*/
btn_style:focused|focus_key{
    ...
}


...... layout.xml

<button class="btn_style" />

```

```
    <div :style="bg_color:bg_color_value">
        <label class="text_css" style="text_decor:underline;color:#ff0000;"  text="hello world" />
    </div>

```
